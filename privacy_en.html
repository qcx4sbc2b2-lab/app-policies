Privacy Policy

Last Updated: December 19, 2025

Introduction

JWB - Japanese Words for Beginners (hereinafter referred to as "the App") is a mobile application designed to support Japanese language learning. This Privacy Policy explains how the App collects, uses, and stores user information.

Developer Information

  • App Name: JWB - Japanese Words for Beginners
  • Contact: japanesewordsforbeginners@gmail.com

Information We Collect

The App stores the following information locally on the user's device:

1. Learning Data

  • Quiz results (correct answers, incorrect answers, accuracy rate)
  • Daily learning records
  • History of incorrectly answered words
  • Words marked as learned
  • Words marked as difficult
  • Words selected for flashcards

2. App Settings

  • Selected learning language (English or Spanish)
  • Quiz time limit settings
  • Dark mode settings
  • Auto-play audio settings

3. App Usage

  • First launch record
  • Language selection completion status

Information We Do Not Collect

The App does not collect the following information:

  • Personally identifiable information (name, email address, phone number, etc.)
  • Location information
  • Contacts
  • Photos or media files
  • Device ID
  • IP address
  • Cookies
  • Any other personal information

Purpose of Data Use

Collected data is used solely for the following purposes:

1. Providing Learning Features

  • Executing quiz functions
  • Recording and displaying learning progress
  • Providing vocabulary notebook features
  • Providing review features

2. Saving App Settings

  • Providing app experience based on user settings

3. Improving Learning Experience

  • Supporting review of incorrectly answered words
  • Displaying learning statistics

Data Storage Location and Method

  • All data is stored locally within the user's device
  • Data is not transmitted to external servers
  • The app can be used without an internet connection
  • Data storage uses standard local storage for iOS/Android (SharedPreferences/UserDefaults)

Data Sharing

The App does not share user data with third parties.

  • We do not use advertising services
  • We do not use analytics services
  • There is no social media integration
  • We do not use external servers or cloud services

Data Security

  • All data is securely stored within the user's device
  • Data is protected by standard iOS/Android security features
  • There is no risk of data leakage through network communication

User Rights

Data Deletion

Users can delete data through the following method:

  • Complete Deletion: Uninstalling the app will delete all data
    • iOS: Long press the app and delete
    • Android: Settings → Apps → JWB → Uninstall

Data Export

Currently, we do not provide data export functionality. All data is stored locally on the device and is under the user's complete control.

Children's Privacy

The App does not intentionally collect personal information from children under 13 years of age. Since the App does not collect personal information, it can be safely used by all age groups.

Audio Features

The App provides audio playback features, but:

  • Does not access the microphone
  • Does not record audio
  • Does not collect user voice data

Changes to Privacy Policy

This Privacy Policy may be updated as necessary. If there are significant changes, we will notify users through the app or App Store/Google Play update information. Changes to this policy will take effect when posted on this page.

Applicable Laws

This Privacy Policy complies with the following laws:

  • Japan's Personal Information Protection Act
  • GDPR (EU General Data Protection Regulation)
  • CCPA (California Consumer Privacy Act)

Since the App does not collect personal information, it is not significantly constrained by these laws.
